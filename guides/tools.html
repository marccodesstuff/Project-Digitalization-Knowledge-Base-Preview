Tools

Unity

Unity is a powerful and versatile game development engine that enables creators to bring their ideas to life. With its intuitive interface and extensive feature set, Unity has become a go-to platform for both indie developers and large studios alike. From 2D mobile games to immersive 3D experiences, Unity provides the tools necessary to craft engaging interactive content across various platforms.

Note that Shattered Eternity will be using Unity 6 for the development of the project. However, the prototype will be developed with Unity 2022 which will allow its developers to prepare the essential game objects ahead of the Unity 6 release on October 17, 2024.

Installation for Windows
To install Unity on Windows, follow these steps:

  1. Visit the official Unity website (https://unity.com/).
  2. Click on the "Get Started" button.
  3. Choose "Individual" plan (free for personal use). You may also apply for the Educational license, which will grant you additional assets (although it is not necessary).
  4. Click "Download Unity Hub".
  5. Run the Unity Hub installer and follow the on-screen instructions.
  6. Once Unity Hub is installed, open it.
  7. In Unity Hub, go to the "Installs" tab.
  8. Click "Add" to install a new Unity version. For this project, you will be using Unity 6 Preview. However, on October 17, 2024, you will need to install the official release of Unity 6 (different from Unity 6 Preview).
  9. Select the latest Long Term Support (LTS) version for stability.
  10. Choose the modules you need (e.g., Windows Build Support).
  11. Click "Install" and wait for the process to complete.
    After installation, you can create new projects or open existing ones through Unity Hub. Remember to regularly check for updates to ensure you have the latest features and bug fixes.

Obsidian.md

Obsidian.md is a powerful note-taking app that lets you build a connected web of your thoughts. Unlike traditional apps, it uses Markdown files for notes, allowing for easy formatting and flexibility. The key feature is the ability to link notes together, creating a visual map of your ideas and making it easy to see connections. With its focus on privacy and customization, Obsidian is a great tool for anyone who wants to organize their knowledge in a non-linear way.

Installation (Windows)

  1. Download the application on Obsidian's website.
  2. Once the download is complete, locate the downloaded .exe and double-click it to run the installer.
  3. Read the license agreement and click "Next" to proceed.
  4. Select the desired installation directory or accept the default location and click "Next."
  5. If you want to add "Open with Code" to the Windows Explorer context menu or configure other settings, check the desired options and click "Next."
  6. Begin the installation process by clicking "Install" to start the installation process.
  7. Once the installation is finished, click "Finish" to close the installer.

Visual Studio Code

Visual Studio Code (VS Code) is a lightweight yet powerful source code editor developed by Microsoft. It has gained immense popularity among developers due to its speed, extensibility, and rich ecosystem of extensions. VS Code supports a wide range of programming languages and frameworks, making it an ideal choice for various development tasks, from web development to data science.

Installation from Website (Windows)
To install Visual Studio Code on Windows, follow these steps:

  1. Visit the official Visual Studio Code website (https://code.visualstudio.com/)
  2. Click on the "Download for Windows" button
  3. Once the installer is downloaded, run the .exe file
  4. Accept the license agreement and click "Next"
  5. Choose the installation location (or leave the default) and click "Next"
  6. Select additional tasks if desired (e.g., adding "Open with Code" action to Windows Explorer context menu) and click "Next"
  7. Click "Install" to begin the installation process
  8. Once installation is complete, click "Finish"

Visual Studio Code should now be installed on your Windows system. You can launch it from the Start menu or desktop shortcut if you created one during installation. After installation, you may want to customize VS Code by installing extensions relevant to your development needs. You can do this by clicking on the Extensions icon in the left sidebar (or press Ctrl+Shift+X) and searching for extensions by name or programming language.

Visual Studio Code Extensions
Here are some of the extensions necessary for the development of Shattered Eternity:

  • Unity - This extension provides IntelliSense, debugging, and other Unity-specific features for C# development in VS Code. It enhances the coding experience for Unity projects by offering code completion, syntax highlighting, and integration with Unity's console output.
  • C# Dev Kit - Offers rich support for C# development, including syntax highlighting, IntelliSense, and debugging capabilities. This is essential for scripting in Unity projects.
  • Live Share - Enables real-time collaborative coding sessions, allowing multiple developers to work on the same project simultaneously. This extension is particularly useful for remote pair programming or team code reviews, enhancing productivity and knowledge sharing within the development team.

Git

Git is a distributed version control system that has become an essential tool for software development and collaboration. Created by Linus Torvalds in 2005, Git allows developers to track changes in their code, manage different versions, and collaborate seamlessly with team members. Its distributed nature means that each developer has a full copy of the project's history, enabling offline work and providing a robust backup system.

Key features of Git include:

  • Branching and merging: Easily create separate lines of development and combine them later
  • Staging area: Review and organize changes before committing
  • Speed and efficiency: Optimized for handling large projects
  • Data integrity: Uses cryptographic hashing to ensure the consistency of project history

Git's popularity has led to the rise of platforms like GitHub, GitLab, and Bitbucket, which provide additional collaboration features and cloud-based repositories. These platforms have further cemented Git's role as an indispensable tool in modern software development workflows.

Installation (Windows)
To install Git on Windows, follow these steps:

  1. Visit the official Git website (https://git-scm.com/)
  2. Click on the "Download for Windows" button
  3. The download should start automatically. If not, click on the manual download link
  4. Once the installer is downloaded, run the .exe file
  5. Review and accept the GNU General Public License
  6. Choose the installation location (or leave the default) and click "Next"
  7. Select the components you want to install (the default selection is usually sufficient)
  8. Choose the default editor used by Git (you can change this later)
  9. Adjust your PATH environment if necessary (the recommended option is usually best)
  10. Choose the SSH executable and HTTPS transport backend
  11. Configure the line ending conversions (the recommended setting is usually best)
  12. Choose the terminal emulator to use with Git Bash
  13. Configure extra options as desired
  14. Click "Install" to begin the installation process
  15. Once installation is complete, click "Finish"

After installation, you can verify that Git is installed correctly by opening a command prompt or Git Bash and typing git --version. This should display the installed version of Git.

GitHub Desktop

GitHub Desktop is a user-friendly application that simplifies the process of working with Git and GitHub. Developed by GitHub, this graphical interface tool provides an intuitive way to manage repositories, commit changes, and collaborate with others without the need for command-line operations. It's particularly useful for those new to version control or developers who prefer a visual approach to managing their projects.

Key features of GitHub Desktop include:

  • Easy repository cloning and creation
  • Visual diff tool for reviewing changes
  • Simplified branching and merging
  • Seamless integration with GitHub.com
  • Built-in merge conflict resolution

GitHub Desktop serves as an excellent bridge between the powerful capabilities of Git and the collaborative features of GitHub, making version control more accessible to a wider range of users.

Installation (Windows)
To install GitHub Desktop on Windows, follow these steps:

  1. Visit the official GitHub Desktop website (https://desktop.github.com/)
  2. Click on the "Download for Windows (64bit)" button
  3. Once the installer is downloaded, run the GitHubDesktopSetup.exe file
  4. The installation will begin automatically. Wait for it to complete
  5. When prompted, sign in to your GitHub account or create a new one
  6. Configure your name and email for Git commits
  7. Choose whether to submit anonymous usage data to help improve the application
  8. GitHub Desktop is now installed and ready to use.

After installation, you can start using GitHub Desktop to clone repositories, make commits, and sync changes with GitHub. The application provides a helpful tutorial for new users to get familiar with its interface and basic operations.

Blender

Blender is a free and open-source 3D creation suite that has revolutionized the world of 3D modeling, animation, and rendering. With its comprehensive toolset and active community, Blender has become a favorite among artists, designers, and even game developers. From creating detailed 3D models to producing stunning visual effects, Blender offers a robust platform for bringing creative visions to life.

Installation (Windows)
To install Blender on Windows, follow these steps:

  1. Visit the official Blender website (https://www.blender.org/)
  2. Click on the "Download" button on the homepage
  3. Select the latest stable version for Windows
  4. Choose between the 64-bit installer or portable version (installer recommended for most users)
  5. Once the download is complete, run the installer
  6. Follow the on-screen instructions in the installation wizard
  7. Choose the installation location (or leave the default) and click "Next"
  8. Select additional options if desired (e.g., creating desktop shortcut) and click "Next"
  9. Click "Install" to begin the installation process
  10. Once installation is complete, click "Finish"

Blender should now be installed on your Windows system. You can launch it from the Start menu or desktop shortcut if you created one during installation. After installation, it's recommended to check for any available updates to ensure you have the latest features and bug fixes. You can do this by going to Help > Check for Updates in the Blender application.

FL Studio

FL Studio, developed by Image-Line, is a digital audio workstation (DAW) that has become a staple in music production. Known for its intuitive interface and powerful features, FL Studio caters to both beginners and professional musicians. The software offers a comprehensive suite of tools for composing, arranging, recording, editing, mixing, and mastering high-quality music.

Installation (Windows)
To install FL Studio on Windows, follow these steps:

  1. Visit the official FL Studio website (https://www.image-line.com/)
  2. Click on the "Download" button in the top navigation menu
  3. Scroll down to find the "Download FL Studio" section
  4. Click on the "Download" button for the Windows version
  5. Once the installer is downloaded, run the .exe file
  6. If prompted by Windows security, click "Run" or "Yes" to allow the installation
  7. In the installation wizard, select your preferred language and click "OK"
  8. Read and accept the license agreement, then click "Next"
  9. Choose the installation type (Full or Custom) and click "Next"
  10. Select the installation location or leave the default, then click "Next"
  11. Choose whether to create desktop and start menu shortcuts, then click "Next"
  12. Click "Install" to begin the installation process
  13. Once installation is complete, click "Finish"

After installation, you'll need to register your copy of FL Studio. If you've purchased a license, enter your account details when prompted. If you're using the trial version, you can explore the software's features for a limited time.

Note that FL Studio offers different editions with varying features. Make sure to choose the edition that best suits your needs and budget when purchasing a license. If you are installing a pirated copy of FL Studio, make sure to purchase it in the future. Support its developers as well.
